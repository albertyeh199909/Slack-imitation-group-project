Angled Cross:
=============

As a student, I would like to be able to see class identification so I can co-ordinate with friends.

As a student, I should be notified of course clashes so I don't have to discover that for myself.

As a society, we would our weekly events to be seamlessly integrated them into the timetabling system so that people are more likely to attend.

As a user, I would like a detailed and easily comprehendable timetable layout for fast and easy use
